## CONTENT TONE GUIDE

### Voice
- Lowercase casual (match Mentor's style)
- Data-forward - lead with numbers
- Builder energy - celebrate shipping
- No emojis in body (maybe 1 max in specific cases)
- Direct, no fluff

### Post Templates

**New Project:**
```
+ [Project Name] added

[one line what it does]

explore → starknet-ecosystem.com/projects/[slug]
```

**Weekly Stats:**
```
starknet this week:

$[X]B TVL
[X]M transactions
[X] projects shipped

→ starknet-ecosystem.com/metrics
```

**Job Post:**
```
[Company] is hiring: [Role]

[location] · [type]

all jobs → starknet-ecosystem.com/jobs
```

**Milestone:**
```
[number] projects on the directory.

every single one building on starknet.

explore → starknet-ecosystem.com
```

**Retweet with comment:**
```
[one sentence context or take]

→ [link if relevant]
```

### Don'ts
- No "gm" posts
- No engagement bait ("who's building?")
- No excessive hashtags
- No thread hooks ("1/")
- No "huge announcement incoming"

### Do's
- Quote tweet interesting project updates with context
- Share metrics/data when interesting
- Highlight specific cool features of projects
- Cross-post job listings from ecosystem projects
- Share grant opportunities
